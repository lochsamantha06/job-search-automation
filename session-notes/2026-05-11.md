# Internship Job Search Automation – Session Notes
**Date:** May 11, 2026

---

## What we built
A fully automated daily pipeline that scrapes Indeed for finance and consulting internships, scores them, finds recruiter contacts, drafts outreach emails with Claude, logs to Google Sheets, and texts a morning digest via Twilio.

## Project location
`C:\Users\lochs\job-search-automation`
GitHub: https://github.com/lochsamantha06/job-search-automation

## How it runs
GitHub Actions cron — every day at **8:00 AM Pacific (15:00 UTC)**, free, no server needed.
Can also be triggered manually: GitHub repo → Actions → Daily Job Scrape → Run workflow.

## Pipeline (in order)
1. Read seen URLs from Google Sheets (deduplication)
2. Scrape Indeed — 13 role queries × 3 cities = 39 searches
3. Filter to new-only jobs
4. Score each job → High / Medium / Low
5. For High/Medium: find recruiter via Hunter.io (LinkedIn fallback) + draft outreach with Claude
6. Append all new jobs to Google Sheets (14 columns)
7. Send SMS digest via Twilio

## Daily routine (5 min)
1. Receive morning SMS
2. Open Google Sheet link in the text
3. Check High priority rows first — outreach draft is in column M
4. Find recruiter via LinkedIn link in column L
5. Copy draft → tweak → send
6. Update column N: "New" → "Applied" or "Skip"

## Things to watch
- If texts stop coming, check GitHub Actions logs — Indeed may be blocking scrapes
- Hunter.io free tier: 25 searches/month — save for High priority jobs
- Twilio free credit (~$15) — top up when depleted
